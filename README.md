# Portu vs Trading 212 Fee Calculator

An interactive calculator comparing the long-term impact of Portu's tiered management fees versus Trading 212 AutoPies (ETF-only).

## Features

- Month-by-month compound growth simulation
- Portu's fee tiers applied dynamically as balance crosses thresholds (500k / 1M / 5M Kč)
- Adjustable monthly contribution and gross return rate
- 10, 20 and 30-year milestone breakdowns
- Interactive area chart with fee tier shown on hover

## Running locally

```bash
npm install
npm run dev
```

## Deployment

Deployed automatically to GitHub Pages via GitHub Actions on every push to `main`.
